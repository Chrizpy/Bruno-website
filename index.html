About Bruno

Bruno is my personal server computer that I have used to lab with during my time as a student at the University of Linköping.

Bruno started out as a system administrator lab companion for a course. The things I learned in that course, I tried on Bruno to see if I understood it. That started a theme of leveraging Bruno to different courses along the way, like web development which was the last course that used Bruno and the reason why Bruno hosts this page today.

Currently Bruno hosts it's own webserver with nginx. It uses Python Flask as the backend. Classic HTML5, CSS and JavaScript is used, we did not use any frameworks like Angular, Vue or React at my university, but I'm interested in learning more about these.
